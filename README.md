# S2C Connect

> Match. Play. Connect. — Ứng dụng kết nối người chơi thể thao.

## Tech Stack

- React 19 + TypeScript + Vite 6
- Tailwind CSS v4
- Zustand, React Router v6, Framer Motion, Sonner, date-fns

## Cài đặt & Chạy local

```bash
npm install
npm run dev
```

## Deploy lên Vercel

1. Push code lên GitHub
2. Vào vercel.com → New Project → Import repo
3. Framework Preset: **Vite**
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Click **Deploy**

File `vercel.json` đã cấu hình sẵn SPA routing.
